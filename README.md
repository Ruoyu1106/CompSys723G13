# COMPSYS 723 – Group 13 Project

This repository contains the implementation and supporting materials for **Group 13's project in COMPSYS 723** at the **University of Auckland**.

The project explores key concepts in **computer systems design and implementation**, focusing on building a practical system that demonstrates core principles such as modular system architecture, implementation of system components, testing, and evaluation.

---

# Project Overview

The objective of this project is to design and implement a working system that demonstrates the application of computer systems concepts learned throughout the COMPSYS 723 course.

The project includes:

- System architecture design
- Implementation of core functionality
- Testing and debugging
- Experimental evaluation and analysis

The repository contains the source code, documentation, and supporting resources used during the project development process.

---

# Key Features

- Modular system implementation
- Clear project structure for maintainability
- Testing and validation of system behaviour
- Experimentation and performance evaluation
- Supporting documentation for system design

Clone the repository:

```bash

```
